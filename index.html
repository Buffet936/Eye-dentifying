Eye Tracking Insider Knowledge

by Manuel Pizana, Armando Mendez, Eddie Ortiz

Introduction

In this project, our main function is to implement machine learning
on the movement of the human eye ball. With this information, the code can find a connection with eye movement and what the gender of the person being tested.

How machine learning is implemented

We conducted our project by analyzing data collected at Texas State University from a sample of over 300 students.
Our study explored whether there is a correlation between saccadic eye movements correlating to an individual's gender by implementing machine learning algorithms.
